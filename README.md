# Operator Blueprints — CAC Calculator (Tool #2)

Fast, minimal CAC Calculator for OperatorOS.  
Static HTML/CSS/JS — zero dependencies — Vercel-ready.

## Features
- Blended CAC
- Paid CAC
- LTV:CAC
- Payback (Orders to Break Even)
- One-click Summary Copy

## Deploy on Vercel
1. Create repo: `operator-cac-calculator`
2. Add files:
   - index.html  
   - styles.css  
   - script.js  
   - README.md  
3. Push to GitHub  
4. Vercel → New Project → Import repo  
5. Framework: **Static HTML**  
6. Build Command: *(empty)*  
7. Output Directory: `/`  
8. Deploy
